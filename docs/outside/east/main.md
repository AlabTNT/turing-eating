# Eating in 东站枢纽

> 杭州东站是杭州市重要的交通枢纽。图灵er们来杭回家，可以试试体验一下枢纽周边的商圈，当然，也少不了美食：

- [杭州东站万象汇](./wanxianghui.md)
- [杭州西子国际](./xiziguoji.md)
- [三花国际大厦](./sanhua.md)
- [环翼城](./huanyicheng.md)
- [港龙悠乐城](./ganglong.md)
- [东站居民圈小吃](./juminqu.md)

> Tips:东站居民圈小吃指的是那些居住在各个大厦、小区的底商中，没有搬进商场的小吃店喵~蝇头小馆有时带来的惊喜会远大于某些商城的！
